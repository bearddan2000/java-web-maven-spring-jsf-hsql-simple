# java-web-maven-spring-jsf-hsql-simple

## Description
A jsf springboot java maven build,
that connects to in memory hsql database.

## Tech stack
- springboot
  - jsf
- maven
  - hsql drivers
  - lombok
  - PostConstruct annotation
- bootstrap
- jquery
- dataTable

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
Available http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
